## Conventions

- One Markdown file per topic.
- Images live in `images/` subfolders alongside their article (e.g. `docs/Assets and Transactions/images/`).
- UK English throughout: organise, favourite, cancelled, analysing.
- Write for a task, not a feature — "How do I...?" not "The X screen".
