# Settings

Settings are accessed via the **Settings** tab in the main navigation.

---

## Sections

| Section | What you configure |
|---|---|
| **Base Currency** | The currency used for all consolidated reports and forecasts |
| **Data Feed** | Automatic market data refresh settings |
| **[Securities](Market Data/Securities.md)** | Add and manage securities for investment tracking |
| **[FX Rates](Market Data/FX Rates.md)** | View stored exchange rates |
| **[Categories](Assets and Transactions/Categories.md)** | Create and manage transaction categories |
| **[Category Rules](Assets and Transactions/Category Rules.md)** | Rules for automatic transaction categorisation |
